\# Contributing



Thanks for your interest in contributing.



This repository is an experimental AI data pipeline prototype

used to evaluate decentralized storage infrastructure such as

Shelby Protocol.



\## Development setup



1\. Clone the repository

2\. Install dependencies



npm install



3\. Run dataset ingestion



node scripts/ingest\_dataset.js



4\. Generate embeddings



node scripts/embed\_dataset.js



5\. Run vector search



node ai/vector\_search.js



\## Areas for contribution



\- dataset ingestion improvements

\- embedding models

\- RAG pipeline improvements

\- Shelby storage integration

\- benchmarking tools
